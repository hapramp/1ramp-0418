# HapRamp Website

Source code for the old HapRamp landing page.

[![Build Status](https://travis-ci.org/hapramp/hapramp-website.svg?branch=master)](https://travis-ci.org/hapramp/hapramp-website)


### Tech Stack

* React (CRA)


### Development

```sh
yarn start
```

### Building for Production

```sh
yarn build
```
